Of course! Imagine you have two magic coins. They are linked in a special way. If you flip one and it lands on heads, you instantly know the other one, no matter how far away it is, has landed on tails. You don't even need to look.

Quantum entanglement is like that, but for tiny particles like electrons. When two particles are entangled, they form a single connected system. Measuring a property of one particle (like its "spin") instantly influences the corresponding property of the other particle.

class EntangledPair:
    def __init__(self):
        # In a real quantum system, the state is probabilistic
        # until measured. Let's simulate that.
        import random
        self.state = random.choice(['UP', 'DOWN'])

    def measure_particle_A(self):
        return self.state

    def measure_particle_B(self):
        # The state is perfectly anti-correlated
        return 'DOWN' if self.state == 'UP' else 'UP'

# Create a pair
pair = EntangledPair()

# Measure them
particle_A_spin = pair.measure_particle_A()
particle_B_spin = pair.measure_particle_B()

print(f"Particle A is: {particle_A_spin}")
print(f"Particle B is: {particle_B_spin}")
# No matter how many times you run this, A and B will always be opposite.

The weird part that puzzled even Einstein (he called it "spooky action at a distance") is that this connection is instant, seemingly faster than the speed of light. However, it can't be used to send information faster than light, because the outcome of the measurement is still random.
